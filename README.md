## Scripts

> all scripts can be run for a particular workspace by appending `--workspace=${name}`

### Build

```bash
$: npm run build
```

### Clean

```bash
$: npm run clean
```

### Lint

```bash
$: npm run lint
```

```bash
$: npm run lint:fix
```

### Dev

> for apps/examples only

```bash
$: npm run dev
```

### Test

> for packages only

```bash
$: npm run test
```
